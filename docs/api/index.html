敏感词管理

# 列出所有敏感词
GET /admin/keywords

# 添加敏感词
POST /admin/keywords
Content-Type: application/json

{"word": "机密项目X", "replacement": "PROJECT_X"}

# 删除敏感词
DELETE /admin/keywords/{id}
